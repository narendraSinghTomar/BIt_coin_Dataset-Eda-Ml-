#  Trader Performance vs Market Sentiment Analysis

## Project Overview
This project analyzes the relationship between market sentiment (Fear vs Greed) and trader behavior/performance using trading data. The goal is to identify patterns that can help in making better trading decisions.

---

##  Dataset
- Bitcoin Market Sentiment (Fear/Greed Index)
- Historical Trader Data (PnL, size, trades, etc.)
